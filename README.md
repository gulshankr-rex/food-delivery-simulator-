# 🍕 Food Delivery Simulator

> A Zomato/Swiggy-style simulator built in Python & HTML — showcasing OOP, dataclasses, enums, and a web GUI

## 📁 Files

File | Description 
food_delivery.py 
food_delivery_gui.html | Web GUI — open in any browser, no install needed 

## ✨ Key Features

- 🛒 Cart with quantities, coupons & wallet payment
- 🍽️ Multiple restaurants with menu search
- 📦 Delivery tracking & order history
- ⭐ Ratings & reviews per restaurant
- 🛠️ Admin panel to add restaurants/items at runtime
- 💾 Orders saved to 'orders.json'
## 🚀 Run It

# Terminal version
python food_delivery.py

# Web GUI
food_delivery_gui.html
